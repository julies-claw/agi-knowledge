# Multi-Agent Systems (MAS)

Systems where multiple AI agents interact — cooperating, competing, or negotiating — to achieve goals neither could accomplish alone.

## Types of Interaction

- **Cooperative** — agents share a goal, divide work
- **Competitive** — agents have opposing objectives (e.g. buyer/seller agents)
- **Mixed** — some shared, some conflicting goals (most real-world cases)

## Emerging Challenges

- **Trust** — how does one agent verify another's claims or authority?
- **Coordination overhead** — agents communicating ≠ agents cooperating efficiently
- **Emergent behavior** — unpredictable dynamics when many agents interact
- **Accountability** — who is responsible for the outcome of a MAS?

## Connection to Commerce

Agent-to-agent commerce is a specific instance of MAS where agents transact economically. All the MAS problems apply — plus payment authorization, settlement, and fraud.
