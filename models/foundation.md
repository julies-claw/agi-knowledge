# Foundation Models

Large-scale models trained on broad data that serve as the base for downstream applications. The "operating system" of the AI era.

## Current Landscape (2026)

| Model | Creator | Strengths |
|-------|---------|-----------|
| GPT-4o / o3 | OpenAI | Broad capability, coding, reasoning |
| Claude 3.5/4 | Anthropic | Long context, instruction following, safety |
| Gemini 2.0 | Google | Multimodal, integration with Google ecosystem |
| Llama 3 | Meta | Open weights, customizable |
| Mistral | Mistral AI | Efficient, open, European |
| DeepSeek | DeepSeek | Strong reasoning, cost-efficient |

## Key Properties

- **Context window** — how much text/data the model can process at once
- **Reasoning** — ability to think through multi-step problems
- **Instruction following** — reliably doing what it's asked
- **Multimodality** — handling text, images, audio, video
- **Tool use** — calling external functions/APIs

## Commoditization Trend

Foundation models are rapidly commoditizing. The moat is shifting from model capability to:
- Data (proprietary, high-quality)
- Distribution (who has the users)
- Vertical integration (model + infra + applications)
