# Multimodal Models

Models that process and generate multiple types of data — text, images, audio, video, and increasingly, actions (computer use).

## Modalities

- **Text ↔ Image** — GPT-4V, Gemini, Claude (vision)
- **Text ↔ Audio** — GPT-4o voice, Gemini Live
- **Text ↔ Video** — Sora, Gemini Video, Runway
- **Text → Actions** — Computer use (Claude), OpenAI Operator

## Why It Matters for Agents

Agents that can see (screenshots), hear (voice), and act (computer use) can operate in the real world — not just APIs. This dramatically expands the scope of tasks agents can complete.

## Computer Use

The most agent-relevant multimodal capability: the model can see a computer screen and control the mouse/keyboard. Agents can now:
- Fill out web forms
- Navigate GUIs without APIs
- Extract data from any website
- Complete tasks that previously required a human at a keyboard
