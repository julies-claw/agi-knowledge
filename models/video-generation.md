# AI Video Generation — State of Art (2026)

Video generation has gone from "interesting demos" to production-quality output in ~18 months.

## Key Players (2026)

| Model | Creator | Notes |
|---|---|---|
| Sora 2 | OpenAI | Improved from Sora 1 (Dec 2023); cinematic quality |
| Veo 3.1 | Google DeepMind | Strong physics, long coherence |
| Kling 3.0 | Kuaishou (China) | Very competitive, rapid iteration |
| Runway Gen-3 | Runway | Creative control, filmmaker-focused |

Source: YouTube comparison (Mar 21, 2026); DEV Community comparison (Feb 2026)

## What Changed (2024 → 2026)

```
2023    Sora released (internal only): 60-second coherent video — shocking
        But: still artifacts, inconsistent physics

2024    Sora public release, Runway Gen-3, Kling 1.0 launch
        Quality: good for short clips, struggles with >30s

2025    Veo 2, Kling 2.0, Sora iterating
        Physics improved, longer clips possible

2026    Sora 2, Veo 3.1, Kling 3.0
        Cinematic quality achievable, longer coherence
        Multiple competing models, rapid iteration
```

## What's Still Hard

- **Long-form coherence**: >2 minutes still degrades
- **Consistent characters**: same person looks different across scenes
- **Text in video**: still unreliable
- **Physics edge cases**: liquids, cloth, hair still show artifacts
- **Cost**: high-quality video generation is expensive per second

## Implications

- Stock footage / B-roll: being disrupted now
- Short-form content creation: democratized
- Film production: tools for pre-visualization, VFX iteration
- Advertising: fast creative iteration
- Long-form narrative film: not yet disrupted
